## About

`hyprdots-devkit` started as a fork of [prasanthrangan/hyprdots](https://github.com/prasanthrangan/hyprdots) and is now maintained as an independent repository.

Our focus:  
- Keep the setup minimal and fast  
- Maintain clean and modern visuals  
- Prioritize a developer-first workflow
